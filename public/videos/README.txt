# APEMS Hero Background Video Assets Directory

This directory is designed to house the optimized, high-quality, cinematic background video files for the APEMS Hero section.

## Suggested Assets:
- `apems-hero-desktop.webm` - Ultra-lightweight WebM format for desktop screens (under 5MB).
- `apems-hero-desktop.mp4` - MP4 fallback format for desktop screens (under 8MB).
- `apems-hero-mobile.webm` - Portrait or center-cropped WebM format optimized for mobile devices (under 2MB).
- `apems-hero-mobile.mp4` - MP4 fallback format for mobile devices (under 3MB).

## Guidelines:
1. Ensure footage is professional, authentic, corporate, and technology-enabled (e.g. AGMs, meetings, live polling, and digital voting).
2. Keep video duration short (15-30 seconds loop) with smooth transitions to ensure fast website load times and zero flashing.
3. The application is configured to automatically load these local assets with a reliable cloud-stream fallback as a robust fail-safe.
