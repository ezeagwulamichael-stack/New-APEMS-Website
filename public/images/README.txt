# APEMS Static Images Directory

This directory houses high-quality fallback and compressed visual assets for the landing page.

## Suggested Assets:
- `apems-hero-poster.webp` - Highly compressed, modern WebP format poster image representing a professional corporate meeting to display while the video loads or when the device has prefers-reduced-motion/low performance.

## Guidelines:
1. Ensure the poster represents technology-enabled corporate assemblies or AGMs with a diverse, professional audience.
2. Maintain high contrast for accessibility standards.
